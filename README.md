# Actividad1-PW2

#Proyecto MERM - PetGym

## Descripcion del proyecto:
Ofrecemos una plataforma web donde loa dueños de las mascotas puedan acceder a guias certificadas y ejercicios. Además, el sitio ofrece un E-Commerce con productos especializados para diferentes animales. El proyecto es fullstack tipo MERN.

## Funcionalidades principales del proyecto

- Biblioteca de guias certificadas por profesionales de la salud
- E-Commerce
- Gamificacion de logros
- Rutinas personalizadas

## Tecnologias a usar:
- Para la base de datos NoSQL, se usara **MongoDB**
- Para la creación de APIs Rest, será **Express**
- La libreria de fronted será **React** por su fácil uso y adaptación
- El entorno de ejecución de js será **Node.js**

## Instalación

### backend
cd backend
npm install
npm start
npm node.js

### fronted - usando ya react
cd fronted
npm create vite@latest my-react-app -- --template react
cd my-react-app
npm install
npm run dev
